# Portfolio Data Repository

Este repositorio contiene todos los datos dinámicos del portfolio de Franco Leon. Los datos se sirven directamente desde GitHub para evitar redeploys innecesarios.

## Estructura de Archivos

- `projects.json` - Información de proyectos del portfolio
- `experience.json` - Experiencia laboral y académica
- `education.json` - Información educativa y formación
- `personal.json` - Información personal y de contacto
- `tech-stack.json` - Tecnologías y habilidades
- `config.json` - Configuración del sistema

## URLs de Acceso

## Cómo Actualizar

1. Edita los archivos JSON en este repositorio
2. Haz commit y push de los cambios
3. Los cambios se reflejan automáticamente en el portfolio sin redeploy

## Formato de Datos

Todos los archivos siguen un esquema JSON específico. Consulta la documentación en el repositorio principal del portfolio para más detalles.
